Tusk

Tusk is a general purpose language with arbitrary precision

Object Oriented

Tusk is an object oriented language, and can create a more scalable and re-usable infrastructure.

Precision

Tusk has arbitrary integer and decimal precision. This means that Tusk is not capped by the 64 bit limit, but rather the system's memory.

Security

Tusk has private and public variables, so you don't need to mess with currying to secure code.

Learning

Tusk has easy and intuitive syntax which makes it easy to learn.
